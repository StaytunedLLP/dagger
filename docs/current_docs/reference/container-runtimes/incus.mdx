# Incus

Dagger can be used with Incus as a local OCI-compatible container runtime.

## Prerequisites

Ensure that the `incus` client can talk to a running Incus daemon on the host:

```console
$ incus info
Server version: 6.x
Server status:  Running
...
```

Incus is available from the [official installation docs](https://linuxcontainers.org/incus/docs/main/installing/). For host and kernel requirements, see the [Incus requirements documentation](https://linuxcontainers.org/incus/docs/main/requirements/).

## Example

```console
$ incus list
NAME    STATE    IPV4    IPV6
$ dagger core version
v0.18.19
$ incus list
NAME                  STATE   IPV4       IPV6
dagger-engine-v0.18.19 RUNNING 10.0.0.12  -
```
